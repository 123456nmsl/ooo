微电影剧本

归土

微电影剧本
文明不是被带走才得以保存，而是被记得，才没有消失。

一、核心概念
片名：《归土》
类型：科幻 / 文明探索 / 人文情感
建议时长：约10分钟
主题表达：当人类失去故土，文明不只是被保存的数据，而是刻在土地、器物、语言、礼仪与记忆中的精神归属。
一句话故事：百年以后，人类重返已经死寂的地球，一支文明回收小队在废墟中寻找遗失的中华文明，却在一座被黄沙掩埋的古老祠堂中，发现了比文物更重要的东西。

二、世界观设定
百年前，地球生态系统彻底崩溃。海洋酸化，森林消亡，动物灭绝，人类被迫启动“星迁计划”，逃往轨道殖民地和外星移民基地。
由于当时技术有限，人类只能优先转移人口、基础科学数据、能源技术和生存设备。大量文明遗产、民间记忆、地方文化、非物质文化遗产被迫遗留在地球。
百年后，人类文明的技术已经高度发达。新一代人类出生在太空，没有见过真正的土地、河流、树木和家园。于是，人类启动“归土计划”，派遣文明回收队返回地球，寻找那些被遗忘在废墟中的文明碎片。

三、主要人物
林舟：女，26岁，文明回收队记录员。出生于火星殖民城，从未见过地球。她冷静、理性，最初认为文明只要被扫描、建模、上传，就算被保存。
陈砚：男，32岁，语言与古文字研究员。祖辈来自地球，他对中华文明有强烈情感，但自己也只是在档案和数据库里认识中国。他相信文明不是数据，而是“被人活过的痕迹”。
AI“玄鸟”：文明回收队的随行智能系统。负责扫描、翻译、建模、修复古遗迹。它理性、精准，却无法真正理解“乡愁”“祭祀”“家族”“根”这样的概念。
老人影像：百年前一位留在地球上的老人。他在最后时刻录下了一段影像，守护着一处地方文明遗存。

四、完整微电影剧本
01 外太空 / 地球轨道 / 远景 / 冷寂
黑暗宇宙中，一颗灰蓝色的星球缓缓转动。它不再是记忆中的蓝色星球。大气层浑浊，陆地大片荒漠化，海洋呈现暗沉的灰绿色。
一艘银白色飞船缓缓接近地球。
旁白 林舟：人类离开地球的第一百零三年，我们终于有能力回来了。
飞船舱内，林舟站在观察窗前，看着远处的地球。她的眼神里没有激动，更多是陌生。
AI玄鸟：归土计划，第七文明回收小队，已进入地球同步轨道。目标区域：东亚大陆，旧中国文明遗存带。
陈砚看着屏幕上逐渐放大的地球影像，轻声说。
陈砚：我们不是回来考古的。
林舟：那是什么？
陈砚没有立刻回答，只是望着地球。
陈砚：是回来认祖。
画面切黑。

02 飞船登陆舱 / 内景 / 冷白光
林舟检查记录设备。她的手腕上投射出任务清单：扫描遗址、回收文字资料、建立三维文明模型、上传至人类文明数据库。
林舟：只要数据完整，文明就不会丢失。
陈砚坐在一旁，手里握着一块破旧的木牌。木牌上刻着一个模糊的汉字：归。
陈砚：你觉得，一个人把母亲的声音录下来，就等于母亲还在吗？
林舟：至少声音没有消失。
陈砚：可声音不知道你冷不冷，也不会叫你回家吃饭。
林舟没有回答。登陆舱剧烈震动，穿过厚重云层。舷窗外，灰黄色的大地扑面而来。

03 地球表面 / 旧中原地区 / 远景 / 荒凉
登陆舱降落在一片废弃平原上。舱门打开，死寂的风卷起沙尘。没有鸟鸣，没有虫声，没有树叶摩擦声。只有风。
林舟踏上地面。她低头看着脚下的尘土，缓缓蹲下，伸手触碰。这是她第一次触摸真正的地球土壤。
AI玄鸟：空气毒性指数：低。生态生命反应：零。建议行动时间不超过四小时。
林舟把一小撮土放进采样管。
林舟：这里就是地球？
陈砚环顾四周。远处，残破的城市轮廓被黄沙覆盖。高楼像折断的骨骼。
陈砚：不。这里曾经是家。

04 废弃城市 / 街道 / 中景 / 静默
两人穿过城市废墟。街道早已被沙土掩埋。路牌断裂，只剩下半个字。便利店的招牌褪色。学校大门倒塌。墙上残留着模糊的标语。
AI玄鸟：检测到汉字残留：仁、礼、家、春。正在建立语义关联。
林舟：仁，古代伦理概念。礼，社会秩序规范。家，血缘共同体。春，季节单位。
陈砚：说得都对。但也都不够。
他们走到一所废弃小学前。教室墙壁半塌，黑板还在。黑板上残留着百年前没有擦干净的粉笔字：明日背诵《静夜思》。
林舟：静夜思？
陈砚走近，看着那行字。他轻声背诵。
陈砚：床前明月光，疑是地上霜。举头望明月，低头思故乡。
林舟沉默。天空灰暗，没有月亮。
林舟：故乡……是地理位置吗？
陈砚：也许是你离开以后，才开始疼的地方。

05 地下遗址入口 / 黄昏 / 悬疑
AI玄鸟扫描到地下异常结构。
AI玄鸟：地下三十七米处存在完整建筑空间。年代推测：二十一世纪以前。结构类型：祠堂、庙宇或地方礼制建筑。
沙地下方，一块石碑露出一角。陈砚清理沙土，石碑上刻着几个字：慎终追远。
AI玄鸟：慎终追远：谨慎对待死亡，追念久远祖先。常见于中华祭祀文化。
林舟：为什么要追念已经死去的人？
陈砚：因为人不是从自己开始的。
两人打开地下入口，进入黑暗。

06 地下祠堂 / 内景 / 暗金色光
地下空间保存得异常完整。这里像一座被时间封存的祠堂。木梁斑驳，牌位蒙尘，墙上有褪色的年画。中央供桌上摆着干裂的瓷碗、香炉和一盏早已熄灭的油灯。
林舟打开照明灯。光照亮了墙上的壁画。画面里有人耕种、祭祖、舞龙、写春联、做灯笼、包饺子、唱戏、送别、迎亲。
这些不是帝王将相，也不是宏大的历史事件，而是普通人的生活。林舟看着这些图像，第一次停下了扫描动作。
林舟：这些……不是重要历史节点。
陈砚：文明本来就不只在博物馆里。
他走到供桌前，轻轻吹去一个木盒上的尘土。木盒中放着一本残破族谱、一只毛笔、一枚铜钱、一小块红布，还有一个老旧影像存储器。
AI玄鸟：检测到可读取影像资料。是否播放？
林舟看向陈砚。陈砚点头。

07 祠堂中央 / 老人影像出现 / 温暖而苍凉
空气中投射出一个老人的全息影像。影像很不稳定，时而闪烁。老人坐在祠堂里，身后也是同样的牌位和香炉。
老人影像：如果有人还能看到这段话，说明你们真的回来了。
老人影像：走的时候太急了，很多东西带不走。大件的带不走，老物件带不走，村口那棵槐树带不走，祠堂也带不走。
老人影像：有人说，这些不重要。重要的是活下去。对，活下去最重要。
老人停顿，眼眶微红。
老人影像：可人活下去以后，总得知道自己是谁。
老人拿起一本族谱。
老人影像：这里面写的不是大人物。都是种地的、教书的、做木匠的、当兵的、开小店的。可他们一代一代，把话传下来，把饭做下来，把节过下来，把名字记下来。
老人影像：孩子，文明不是你们带走了多少金银器物，也不是数据库里有多少条记录。
老人影像：文明是一个人知道，自己不是孤零零来到世上的。
老人影像：如果你们回来了，替我们上一炷香吧。不为神仙，也不为保佑。就是告诉他们，后人还记得。
影像中断。祠堂陷入安静。

08 祠堂 / 内景 / 情感转折
林舟站在原地，久久没有说话。AI玄鸟打破沉默。
AI玄鸟：资料完整度较高。建议立即扫描上传。此处物理结构不具备长期保存条件。
林舟看着供桌上的香炉。
林舟：玄鸟，生成一炷香。
AI玄鸟：任务清单中无祭祀行为。
林舟：新增任务。
AI玄鸟：无法理解该行为的功能意义。
林舟：我也不完全理解。
她拿起一支残破的香。香早已不能燃烧。陈砚从装备包里取出微型热源，小心点燃。一缕微弱的烟升起。
这是死寂地球上，百年后重新升起的第一缕香烟。林舟和陈砚站在供桌前。两人没有跪拜，只是安静地低头。
林舟：我们回来了。
陈砚：我们还记得。
香烟缓缓升向黑暗的屋顶。AI玄鸟开始记录这一幕。
AI玄鸟：正在记录未知文明行为：归来者向先人表达记忆确认。
林舟：不叫未知行为。叫祭祖。

09 祠堂壁画 / 文明复苏视觉段落
随着扫描设备启动，整个祠堂被柔和的金色光线覆盖。AI玄鸟将壁画、族谱、文字、器物、影像逐一数字化。
但这一次，不再是冰冷的数据扫描。壁画仿佛被光唤醒：有人在清晨贴春联；有人在院中包饺子；孩子举着灯笼奔跑；老人坐在门口讲故事；戏台上锣鼓响起；纸鸢飞上天空；毛笔落在宣纸上；龙灯在人群中游动；一双手把饭碗递给另一双手。
这些影像像记忆一样在祠堂中流动。林舟站在光影中央，第一次露出柔软的神情。
林舟旁白：我曾以为，文明是一种可以被复制的资料。
林舟旁白：后来我才明白，文明也是一声称呼，一次等待，一盏灯，一顿饭，一条回家的路。

10 地球表面 / 黎明 / 开阔
林舟和陈砚走出地下祠堂。远处天边出现微弱的晨光。荒凉的大地依旧没有生命，但光线比之前温暖。
林舟回头看向祠堂入口。她将一枚小型文明信标插入土地。信标启动，投射出一行汉字：中华文明遗存点001：归土祠堂。
AI玄鸟：是否将该遗址标记为可迁移文明数据？
林舟：不。标记为人类精神原乡。
陈砚看着她，微微笑了。飞船远处降下牵引光束。林舟准备离开，却忽然停住。
她从采样管中倒出一点土，放进一个透明吊坠里，挂在胸前。
陈砚：带走样本？
林舟：带一点家。

11 飞船返航 / 地球轨道 / 结尾
飞船离开地表，升入云层。地球再次出现在窗外。林舟坐在舱内，开始录制任务报告。
林舟：归土计划，第七文明回收小队，第一次地表任务完成。
林舟：本次任务回收文物三百七十二件，影像资料六十四组，文字资料一千九百二十七页，建筑结构模型一处。
她沉默片刻，关闭官方报告界面，重新打开私人记录。
林舟：私人记录，林舟。今天，我第一次知道，地球不是旧档案里的母星编号。
她望向远处的星球。
林舟：它叫故乡。
画面中，飞船渐渐远离。地球静静悬在黑暗宇宙中。
字幕：文明不是被带走才得以保存，而是被记得，才没有消失。
画面切黑。

五、影片风格建议
前半段：冷灰色、低饱和、机械感、荒凉、静默，突出地球死亡后的空旷。
进入祠堂后：暗金色、尘埃光、木质纹理、东方礼制空间，形成强烈的文明温度。
结尾：灰蓝色地球与微弱晨光结合，既不完全乐观，也保留希望。

六、核心情绪递进
• 陌生：新一代人类不认识地球，只把它当作任务现场。
• 震撼：他们在废墟中发现文明并不是宏大符号，而是普通人的生活。
• 理解：通过老人影像，他们明白“文明”不是数据库，而是人与祖先、土地、记忆之间的连接。
• 归属：林舟从“记录者”变成“后人”，完成情感上的认祖归宗。

七、宣传语提炼
1. 百年以后，人类重返地球，寻找被遗忘的自己。
2. 我们逃离过故乡，却从未真正离开文明。
3. 当最后一缕香烟升起，失落的文明终于被后人记起。
4. 文明不是数据，是有人还记得回家的路。
5. 带走一捧土，也带回一个名字：故乡。
